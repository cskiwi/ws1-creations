EXAMEN SERVERSIDE WEBSCRIPTEN — 2013-2014 — Groep 2

Dit document vormt een aanvulling op hetgeen beschreven in examen.html

Opgavebestanden

Gegeven is één .zip bestand met daarin enkele HTML templates. We onderscheiden 3 pagina's:

  • Een overzichtspagina index.html met daarop een lijst van filmcovers, in oplopende titel-volgorde gesorteerd.
  • Een aanmeldformulier login.html waar een gebruiker zich kan aanmelden.
  • Een uploadformulier add.html waar een aangemelde gebruiker een cover kan uploaden.

Van elke pagina werden enkele varianten (met/zonder foutmeldingen, aangemeld/niet aangemeld, met/zonder covers) toegevoegd, doch zijn dit in essentie steeds dezelfde pagina's.

Aanvullend vind je eveneens een dump.sql bestand met daarin een database sws_groep2 die de informatie getoond op de statische pagina's omvat.

De database bestaat uit drie tabellen:

  • movies - Omvat alle films op de site, gelinkt aan een user die de cover online plaatste (veld user_id) en een genre (veld genre_id).

  • users - Omvat alle gebruikers die covers kunnen online plaatsen. De wachtwoorden (veld password) zijn met crypt() geëncrypteerd en is steeds Azerty123.

  • genres - De genres waartoe een film kan behoren. Een film is steeds aan één genre toegekend.

De gebruikte covers vind je in de files/ map. Kopieer deze mee in je project. Daarnast tref je ook nog twee afbeeldingen (een .png en een .jpg) aan die je voor de uploadfunctionaliteit kan gebruiken (deze afbeeldingen hebben reeds de correcte afmetingen).

Uitwerking

De uitwerking doe je volgens de instructies in het bestand examen.html. In totaal zal je 4 PHP pagina's moeten maken:

  • de overzichtspagina index.php
  • de aanmeldpagina login.php
  • de afmeldpagina logout.php
  • de toevoegpagina add.php

Versnij de gegeven HTML code zodat deze in je .tpl files bruikbaar worden. Wijzig echter niks aan de opbouw van de HTML, maw: het is niet toegestaan om HTML elementen toe te voegen, of te verwijderen. De CSS en JavaScript laat je eveneens ongewijzigd. Aan de datasestructuur wordt evenmin geraakt.

Puntenverdeling 'Functioneel'

  • Overzichtspagina ( /4)

    • Weergave films (titel, naam uploader, genre), in oplopende titel-volgorde
      • De vergroting in de lightbox gebeurt via JavaScript. Alle code hiervoor is voorzien; je hoeft enkel het juiste .js bestand in de HTML code invoegen.
    • Weergave genres bovenaan met doorklik + filter op aangeklikte genre
    • Merk op: er zijn genres die geen films omvatten. Verhinder eveneens het bereik van een niet bestaande genre.

  • Session Control & Cookies ( /3)

    • Persistentie/formchecking bij aanmeldformulier
      • Persisteer eveneens de username via een cookie indien aanmelden gelukt is.
    • Toevoegpagina mag enkel bereikbaar zijn mits aanmelden + bewaren user_id bij een nieuwe upload.
    • Aanpassing menu rechtsboven op basis van aanmeldstatus

  • Toevoegen afbeelding ( /3)

    • Persistentie/formchecking uploadformulier
      • Vergeet extra het juiste .js bestand in de HTML code in te voegen.
      • Je mag er van uitgaan dat een afbeelding reeds met de juiste afmetingen geupload wordt, herschaling hoef je niet in te bouwen. Je dient uiteraard wel de extensie te controlen.
    • Invoegen record in databank
    • Correct benoemen afbeelding op schijf (steeds files/covers/ gevolgd door het insert id, een ., en de extensie van het originele bestand)
